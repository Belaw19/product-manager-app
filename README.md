# Flutter Product Manager App

A Flutter CRUD application using Provider state management and http package.

## Features

- Create Product
- Read Products
- Update Product
- Delete Product
- Loading Indicator
- API Integration

## Packages Used

- provider
- http

## API Used

https://fakestoreapi.com/products
